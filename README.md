# MERN AUTHENTICATION
This project puts a focus on implementing authentication in mern stack applications using jsonwebtoken and oauth using passport.

## Running locally
Clone this repo on your device to get started.

Navigate to where you want to store the code

`cd ${path}`

Then run the following command

`git clone https://github.com/StevenMODIMO/mern-auth`

Navigate to the cloned repo

`cd mern-auth`

Inside the folder there are two main folder one for the frontend and backend code.
`frontend` and `backend`

To kickstart the backend development server :-

`cd backend`

`npm install`

`npm run dev`

This will run the backend server on `http://localhost:4000`

To kickstart the frontend development server :-

`cd frontend`

`npm install`

`npm run dev`

This will run the frontend server on `http://localhost:5173`
